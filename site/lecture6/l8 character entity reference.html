HTML entity是一種用來表示HTML中的預留字元或特殊字元的方法。預留字元是指那些在HTML中有特殊意義的字元，例如小於號（<）和大於號（>），它們會被瀏覽器解釋為標籤的開始和結束。特殊字元是指那些不屬於文件的字元集的字元，例如分號（￠）和歐元符號（€）。如果想要在HTML中正確地顯示這些字元，我們必須使用HTML entity來替換它們。 HTML entity的格式是&entity_name;或&#entity_number;，其中entity_name是一個預定義的名稱，entity_number是一個十進制或十六進制的數字。例如，要顯示小於號，我們可以寫<或<或<。要顯示歐元符號，我們可以寫€或€或€。 HTML entity的好處是，它們可以避免瀏覽器對預留字元的誤解，並且可以顯示一些不常用或難以輸入的字元。HTML entity的缺點是，它們會增加文件的長度，並且可能不被所有的瀏覽器或字元集支援。
